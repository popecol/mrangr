# Example Simulated Community Output

A `sim_com_results` object containing results of a 20-step simulation of
a 4-species community.

The simulation was generated using the [`community_eg`](community_eg.md)
object.

This object can be accessed via the
[`get_simulated_com`](get_simulated_com.md) function.

## Format

An object of class `sim_com_results` from the `mrangr` package generated
using [`sim_com`](sim_com.md).

## Source

Data generated in-house to serve as an example
